## 🧠 安装 SenseVoice  
1. **克隆仓库**:  
   ```bash
   git clone https://github.com/FunAudioLLM/SenseVoice.git
   ```
2. **进入目录**:  
   ```bash
   cd SenseVoice/
   ```
3. **安装依赖**:  
   ```bash
   pip install -r requirements.txt
   ```

---

## 🔑 申请 Telegram API  
1. **访问**: [my.telegram.org](https://my.telegram.org/)  
2. **配置**:  
   - 设置 **应用名称** 和 **短名称**  
   - 选择 **Desktop**  
3. **获取凭据**:  
   - 保存 **api_id** 和 **api_hash**  
4. **更新代码**:  
   - 在 `app.py` 中替换 **api_id**、**api_hash** 和 **phone**
