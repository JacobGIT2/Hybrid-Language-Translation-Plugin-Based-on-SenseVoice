🌟 SenseVoice 混合语言翻译插件

---
[English](./Readme.md)

📖 背景





- 模型: SenseVoice



- 通信: Telegram (绕过公共服务器应用层端口限制)



- 功能: 支持混合语言翻译



- 平台: iOS 快捷指令



🚀 工作流程





- 📱 iOS: 点击快捷指令 → 录音 → 发送至 Telegram 机器人



- 🖥️ 服务器: 接收音频 → 推理翻译



- 📤 服务器: 发送结果至 Telegram 聊天



- 📲 iOS: 接收结果 → 弹窗 (翻译为中文) → 复制到剪贴板 (原文)



⚙️ 要求





无额外硬件需求，仅需运行 SenseVoice 模型
